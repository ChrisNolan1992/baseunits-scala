org.sisioh.baseunits.scala.time

CalendarInterval

class CalendarInterval extends Interval[CalendarDate] with Serializable

期間（日付の区間）を表すクラス。

限界の表現には CalendarDateを利用する。 生成する期間の開始日と終了日は期間に含む（閉じている）開区間を生成する。
