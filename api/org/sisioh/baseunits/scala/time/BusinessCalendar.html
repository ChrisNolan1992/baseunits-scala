class BusinessCalendar extends AnyRef

営業日カレンダー。

営業日と非営業日を判定する責務を持つ。非営業日とは休日（祝日）及び週末（土日）を表し、営業日とは非営業日でない日を表す。 週末は休日ではないが、週末かつ休日は休日である。
