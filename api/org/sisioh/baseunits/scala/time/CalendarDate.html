CalendarDate

class CalendarDate extends Ordered[CalendarDate] with Serializable

カレンダー上の特定の「年月日」を表すクラス。

java.util.Dateと異なり、時間の概念を持っていない。また、 TimePointと異なり、 その日1日間全ての範囲を表すクラスであり、特定の瞬間をモデリングしたものではない。
